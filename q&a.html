Question-1: What are the different ways to select an element in the DOM?

Their are many way to select an element in DOM. like getElementById, getElementByClassName, getElementByTagName, querySelector, querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML use for sets HTML content including tags ; innerText use for sets any text ; textContent use for sets all content including hidden text

Question-3: What is event delegation in the DOM?

Event delegation হল JavaScript-এ একটি টেকনিক যেখানে একটি parent element কে ইভেন্ট লিসেনার যোগ করে তার সমস্ত চাইল্ড এলিমেন্টের ইভেন্টগুলি ম্যানেজ করেন

Question-4: What is event bubbling in the DOM?

Bubble Up হল DOM এ ইভেন্ট হ্যান্ডলিং এর একটি প্রক্রিয়া, যেখানে কোন html এলিমেন্টে ইভেন্ট ঘটলে (যেমন ক্লিক করলে) সেই ইভেন্টটি DOM ট্রি উপরের দিকে (প্যারেন্ট এলিমেন্ট এর দিকে) ছড়িয়ে পড়ে

Question-5: How do you create, add, and remove elements using JavaScript?

Create Element with document.createElement('tagName'); add an element with appendChild(); remove an element with document.getElementById('idName').remove();
